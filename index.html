Procedimentos (Runbooks)

Alterar senha / Esqueci login

Quando usar

Quando o cliente solicita a alteração de senha ou recuperação de login.

Pré-requisitos

• Acesso ao painel administrativo
• Link do campeonato, nome de usuário ou e-mail para localizar no sistema
• Confirmação dos dados do cliente (nome, e-mail ou usuário)
• Pergunte qual senha deseja colocar
• Sem referência? Confirme se ele criou um campeonato

Passos

  1. Acesse o painel administrativo do campeonato
  2. Vá em “Usuário” → “Nova senha” → “Alterar senha”
  3. Após alterar, deve aparecer “OK”
  4. Envie usuário e senha ao cliente
  5. Login Google, ou Apple: Cadastro feito com o Google/Apple, não tem senha. Siga a sugestão de mensagem.
  6. Login Facebook: crie novo usuário e senha; peça e-mail e a senha desejada

Escalar se...

• Erro durante o procedimento
• Você não conseguiu achar a conta do cliente

Mensagem ao cliente

"Senha alterada com sucesso! Você pode acessar com
Usuário: [NOME_DE_USUARIO]
Senha: [SENHA]. Qualquer dúvida, fico à disposição."
"Seu login via Facebook foi desativado. Criei um novo usuário e senha e transferi seus dados.
Usuário: [NOME_DE_USUARIO]
Senha: [SENHA]. Por favor, verifique se está tudo certo."
"Nesse caso, como você fez login com uma conta Google/Apple, não tem como a gente editar a senha. Basta você selecionar a opção de "Entrar com o Google/Apple" e escolher seu e-mail."
"Nesse caso, como você fez login com uma conta Apple, não tem como a gente editar a senha. Basta você selecionar a opção de "Entrar com a Apple" e escolher seu e-mail.""

Transferir dados de uma conta para outra

Quando usar

Quando o cliente precisa mover todos os dados da conta antiga para uma nova conta.

Pré-requisitos

• Acesso ao painel administrativo
• Identificar as duas contas (antiga e nova) via link/ID/usuário/e-mail
• Garantir que a conta nova esteja vazia (sem campeonatos ou dados ativos)

Passos

  1. Acesse o painel administrativo do campeonato
  2. Na conta nova, copie o link/ID
  3. Volte para a conta antiga
  4. Cole o link/ID no campo “Novo usuário”
  5. Clique em “Transferir dados”
  6. Após transferir, deve aparecer “OK”

Observações

Se aparecer “Error” ao transferir, a conta nova já possui dados (ex.: campeonato ativo). A transferência só funciona para contas vazias.

Escalar se...

• Erro durante o procedimento
• Você não conseguiu achar a conta do cliente

Mensagem ao cliente

"Transferência concluída! Movi todos os seus dados da conta antiga para a nova. Acesse com o novo usuário e confira se está tudo certo. Qualquer divergência, me avise que eu ajusto pra você."
"Tentei fazer a transferência, mas a conta nova já possui dados e o sistema retornou “Error”. Para concluir, preciso que a conta nova esteja vazia. Posso criar uma conta limpa para você ou usar outra conta sem dados."

Gerar boleto bancário

Quando usar

Quando o cliente solicita um boleto bancário para pagamento.

Pré-requisitos

• Acesso à conta bancária da empresa
• Dados do cliente (nome, CPF/CNPJ, CEP, endereço)

Passos

  1. Acesse a conta bancária → Cobrar/Receber → Nova Cobrança
  2. Selecionar cliente / Cadastrar Cliente
  3. Pesquisar pelo CPF/CNPJ na lupa e confirmar
  4. Gerar o boleto, baixar e enviar ao cliente

Escalar se...

• “Nova Cobrança” indisponível
• Erro ao gerar o boleto
• Dados divergentes do informado

Mensagem ao cliente

"Obrigado por aguardar! Aqui está o seu boleto para pagamento. Qualquer dúvida, fico à disposição."

Responsável: Felipe | Última revisão: 24/09/2025
